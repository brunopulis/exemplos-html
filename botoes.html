Botões, o que são e para que servem?

O Elemento HTML button representa um botão clicável.

Categorias de conteúdo Conteúdo de fluxo, conteúdo fraseado, Conteúdo iterativo, listado, rotulável, e elemento enviável  associado a formulário, conteúdo palpável.


Tipos de Botões

Existem três categorias de botões no HTML, quais sejam:

submit:
O botão envia os dados do formulário para o servidor. Esse é o padrão se o atributo não for especifidado, ou se o atributo é dinamicamente mudado para um valor vazio ou inválido.
reset:
O botão restaura todos os controles para seus valores iniciais.
button:
O botão não possui comportamento padrão. Ele pode ter scripts do lado do cliente associado com os eventos do elemento, no qual são acionados quando o evento ocorrer.

Assim, podemos concluir que, as duas primeiras categorias de botões são específicos para uma determinada ação, enquanto a terceira categoria de botão, qual seja, Button ou botão de ação, pode ser adaptado para infinitas ações de acordo com a vontade do programador, como por exemplo: botões de alternância, botões de expandir e recolher ou apenas botões de ação simples.

Botões de alternância

Um botão de alternância é semelhante a uma caixa de seleção porque está selecionado ou não selecionado, e cada botão de alternância pode operar independentemente de todos os outros botões de alternância.
Na verdade, uma caixa de seleção pode ser uma alternativa aceitável para um botão de alternância.
